===============================================
SEND GOLD TO PLAYER MOD v2.0.0
For Crusader Kings III 1.17.x
===============================================

DESCRIPTION:
This mod allows you to send specific amounts of gold 
to other human players in multiplayer games.

FEATURES:
- Send preset amounts: 1, 5, 10, 25, 50, 100, 250, 500, 1000 gold
- Send ALL your gold option
- Only shows for human players (not AI)
- Bilingual: English and Spanish

HOW TO USE:
1. Right-click on another player's character
2. Select "Send Gold" from the interaction menu
3. Check the amount(s) you want to send
4. Click Accept

TIP: To send exact amounts like 177 gold, combine multiple sends:
     100 + 50 + 25 + 1 + 1 = 177

INSTALLATION:
1. Extract this folder to:
   Documents\Paradox Interactive\Crusader Kings III\mod\
2. Create a .mod file or use the Paradox launcher to detect it
3. Enable in the launcher before starting a game

MULTIPLAYER NOTE:
All players must have the same version of this mod installed.
